# MyTorch library

A custom PyTorch like toolkit for deep learning built using Python and Numpy

Includes:
* Loss functions
* Optimizers
* Linear layer
* Convolutional layer
* Recurrent layer
* Batchnorm2D
* Meanpool2D
* Maxpool2D
* Implementation of MLPs, CNNs, LSTMs, and GRUs
* and other PyTorch utilities
